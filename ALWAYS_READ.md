# 항상 읽기

매 세션을 시작할 때마다 아래 문서를 먼저 확인해 프로젝트 컨텍스트, 스펙, 워크플로가 일관되게 유지되도록 하세요.

1. `AGENTS.md` - 기여 가이드와 저장소 운영 규칙.
2. `docs/README.md` - 스펙, 계획, 할일, 이슈 문서의 위치를 설명하는 문서 인덱스.
3. 각 인덱스에서 가장 최근 항목(특히 `docs/specs/doc-editor.md`, `docs/planning/development-plan.md`)을 확인해 요구사항과 진행 현황을 최신 상태로 파악.

기준 문서가 변경되거나 필수 참고 문서가 추가되면 이 목록도 함께 갱신하세요.
