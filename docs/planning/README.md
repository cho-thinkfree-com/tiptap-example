### 가이드라인
- 짧고 핵심적인 항목 위주로 작성하세요. 한 화면을 넘는 경우 `milestone-01.md`처럼 문서를 분리하고 이 표에 링크를 추가합니다.

- 마일스톤이 완료되면 해당 섹션을 보관용 파일로 이동하고, 이 표에서는 `Superseded`로 표시합니다.
- 상태 표시는 `docs/tasks/` 폴더의 작업 파일과 동기화합니다.
